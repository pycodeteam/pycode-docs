# PyCode

PyCode is an independent initiative focused on building data platforms, web applications and digital tools for data-driven projects.

## Activities

- Data engineering (data collection and processing)
- Dataset structuring and management
- Web application development
- Data visualisation

## Projects

### Lekkantu
Nutrition and food product data platform.

### SunuGoal
Sports data and statistics platform focused on Senegalese football.

### Kurel Facts
Cultural data structuring and digitalisation.

## Documentation structure

This repository contains public documentation organised as follows:

- `institutionnel/` → vision, positioning and activities  
- `projets/` → project documentation  
- `datasets/` → data structures and schemas
