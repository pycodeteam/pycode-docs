# Lekkantu Dataset Structure

This document describes the data structure used for Lekkantu.

## Core fields

- Category
- Brand
- Sub-brand
- Barcode
- Country of origin
- Manufacturer / importer
- Packaging

## Ingredients

- Ingredients list
- Allergens list

## Nutritional values

- Energy
- Proteins
- Carbohydrates
- Sugars
- Fats
- Saturated fats
- Salt
- Sodium
- Other nutrients

## Notes

- Observations
- Additional comments

## Approach

Data is collected and progressively standardised to ensure:

- consistency
- comparability
- usability for analysis and applications
