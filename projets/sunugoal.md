# SunuGoal

SunuGoal is a data project focused on Senegalese football.

## Objective

- Track player performance
- Structure competition data
- Provide reliable football statistics

## Data

- Players
- Matches
- Minutes played
- Goals, cards, injuries
- Competitions

## Scope

- Senegal Ligue 1
- National team
- Senegalese players abroad

## Approach

- Data collection from local sources
- Structured datasets
- Progressive enrichment over time

## Vision

SunuGoal aims to create a structured football data ecosystem to:

- support analysis
- highlight local talent
- enable future platforms and dashboards
